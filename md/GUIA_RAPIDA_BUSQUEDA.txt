┌─────────────────────────────────────────────────────────────────────────────┐
│                      📄 SECCIONES DEL PERFIL                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  ┌─ TICKETS CREADOS                                                        │
│  │  • Todos los tickets que creó el usuario (abiertos)                    │
│  │  • Info: Número, título, estado, responsable, fecha                   │
│  │  • Buscador: Por número, título o descripción                         │
│  │  • Paginación: 9 tickets por página                                   │
│  │                                                                         │
│  ├─ MENCIONADO EN                                                          │
│  │  • Tickets donde fue mencionado en comentarios (@usuario)              │
│  │  • Info: Número, título, estado, solicitante, fecha                   │
│  │  • Buscador: Filtra dentro de tickets con menciones                   │
│  │  • Paginación: 9 tickets por página                                   │
│  │                                                                         │
│  └─ ACTIVOS                                                                │
│     • Todos los activos asignados al usuario                              │
│     • Info: RFK, tipo, fabricante, ubicación                              │
│     • Buscador: Por nombre o número de activo                             │
│     • Paginación: 9 activos por página                                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
